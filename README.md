# SithCare Prototypes Repository
 
> UI/UX design artefacts and prototypes for the **SithCare Mental Health Channelling Platform**
>
> Supporting the **System Design phase** of **SENG 31242 – System Design Project** at the University of Kelaniya.
 
---
 
## Overview
 
This repository manages all prototype and design artefacts across the SithCare platform, including:
 
- Low-fidelity wireframes
- Dashboard mockups
- User flow diagrams
- Figma exports
- Prototype design assets
- UI review materials
---
 
## Repository Structure
 
```text
sithcare-prototypes/
│
├── README.md
├── CONTRIBUTING.md
├── .gitignore
├── wireframes/
├── figma-exports/
├── user-flows/
└── dashboard-mockups/
```
 
### `wireframes/`
 
Low-fidelity wireframes for all major system modules:
 
- Patient dashboard
- Doctor dashboard
- Clinic administration
- Appointment booking
- Emergency support
- Community support system
### `figma-exports/`
 
Exported design assets from Figma, including:
 
- PNG files
- SVG files
- PDF exports
### `user-flows/`
 
Navigation and interaction flow diagrams, including:
 
- Appointment booking flow
- Emergency support flow
- Telemedicine consultation process
### `dashboard-mockups/`
 
High-level dashboard mockups for:
 
- Patients
- Doctors
- Clinic administrators
- System administrators

## Branching Strategy
 
| Branch | Purpose |
|---|---|
| `main` | Stable approved version |
| `draft/` | Active prototype work |
| `fix/` | Corrections and revisions |
 
**Examples:**
 
```
draft/patient-dashboard-wireframe
draft/emergency-support-ui
fix/12-login-layout
```
